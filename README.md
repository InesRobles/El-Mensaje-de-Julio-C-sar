# Cifrado César

## 🔎Objetivo🔍
- Desarrolla un programa en Java que permita cifrar y descifrar un mensaje utilizando el algoritmo de cifrado César.

## 🧩Requisitos🧩
- El programa debe solicitar un texto y una clave (un número entero que representa el desplazamiento).
- Cada letra del texto original debe desplazarse X posiciones en el alfabeto. Por ejemplo, si la clave es 3:
-- 'A' se convierte en 'D'.
-- 'B' se convierte en 'E'.
- Si el desplazamiento sobrepasa la 'Z', debe volver a empezar desde la 'A' (aritmética modular).
- Extra: El programa debe mantener los espacios y caracteres especiales sin cifrar.

## 💻Tecnologías💻
- Java
- IDE: Eclipse

## 🚨 ¿Cómo ejecutarlo? 🚨
- Abrimos el archivo java en Eclipse IDE.
- Pulsamos la flecha verde que se encuentra en la barra superior.
- Aparece la consola, ya puedes jugar.
